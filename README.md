## Description

現代では、どうしてもCDを個人の趣味嗜好基準で選択してしまい、
興味の幅を広げるのが難しくなっているかと思います。
SHAKA-SHAKAは、店頭で体験するような「音楽との偶然の出会い」をネット上で実現できるサービスです。

## Features

- 商品ランダムサジェスト
- 商品を購入する
- 送付先を複数登録し、購入時に選択する
- 注文履歴を確認・キャンセルする
- お問い合わせメールを送る
- 会員登録・編集・退会
- 商品名検索
- 在庫終了ステータス表示
